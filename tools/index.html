Research Tools

I develop and maintain open-source tools to support academic research and information discovery. These tools are designed to make literature search and management more efficient and intelligent.

üîç Academic Paper Finder

An intelligent academic paper discovery tool powered by AI and multiple data sources. Perfect for researchers who need continuous monitoring of new publications in their field.

Key Features:

  • Multi-source search (arXiv, PubMed, Semantic Scholar, OpenAlex)
  • AI-powered relevance analysis and summarization
  • Direct Zotero integration for library management
  • Automated continuous search and monitoring
  • Configurable search parameters and filters

Use Case: Download and run locally for continuous paper monitoring, automatic storage, and library integration.

Repository: github.com/CGutt-hub/paper-finder

Download: Available as a Python package - see repository for installation instructions.

üöÄ Custom Search Engine (SearXNG)

A privacy-focused, self-hosted search engine optimized for academic research. Built on SearXNG with custom configuration prioritizing scholarly databases and scientific search.

Key Features:

  • Multi-source search (arXiv, PubMed, Google Scholar, Semantic Scholar, CrossRef, and 20+ more)
  • Privacy-first: no tracking, no logging, no ads
  • Custom theme matching the 5ha99y design
  • Search shortcuts for quick access to specific databases
  • Dockerized for easy deployment

Use Case: Your own personal search engine that you control. Deploy to cloud (free tiers available) or run locally on your network. Perfect for researchers who want privacy + comprehensive academic search.

Deployment: Self-hosted (requires Docker)

  • Quick Start: See deployment guide
  • Free hosting options: Oracle Cloud, Google Cloud, Fly.io
  • Local cluster: Migrate to your own hardware later

Repository: github.com/CGutt-hub/searxng5ha99y
