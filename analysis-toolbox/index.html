Analysis Toolbox

Python toolbox for robust processing, feature extraction, and statistical analysis of multimodal physiological data.

What is the Analysis Toolbox?

The Analysis Toolbox emerged from hands-on research in cognitive neuroscience, specifically developed to tackle the challenges of processing and analyzing complex physiological signals. When working with multimodal data like EEG, fNIRS, and ECG simultaneously, researchers need reliable pipelines that handle preprocessing, artifact removal, feature extraction, and statistical modeling in a coherent workflow.

This toolbox provides exactly that: a collection of validated signal processing methods and statistical analysis routines built on top of established scientific Python libraries (NumPy, SciPy, MNE, Statsmodels). It automates common analysis tasks while maintaining flexibility for custom workflows, helping researchers move from raw physiological recordings to interpretable results more efficiently.

Whether you're studying neural-autonomic synchrony, investigating brain-computer interfaces, or analyzing embodied cognition experiments, the toolbox offers tested building blocks that integrate smoothly into your research pipeline. All methods are documented and follow open science principles, supporting reproducibility and collaborative development.

Key Features

  • Multimodal Signal Processing — Unified preprocessing for EEG, fNIRS, ECG, PPG, and EDA recordings
  • Feature Extraction — Automated computation of time-domain, frequency-domain, and complexity features
  • Statistical Analysis — Built-in routines for univariate and multivariate statistical modeling
  • Neural-Autonomic Integration — Methods for quantifying synchrony between brain activity and autonomic signals
  • Quality Assurance — Robust artifact detection and data quality metrics
  • Open Source — MIT licensed, fully documented, and designed for extension

Download & Documentation

The toolbox is available as an open-source Python package on GitHub. Full documentation, installation instructions, and example workflows are included in the repository.
